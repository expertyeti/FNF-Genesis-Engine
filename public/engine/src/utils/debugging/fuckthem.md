Activación Maestra
Consola: window.toggleEngineDebug() (Requiere que el valor en el registry sea true o lo fuerza localmente).

Jugabilidad
Alt + B : Activar/Desactivar Autoplay (Botplay).

Alt + G : Activar/Desactivar Ghost Tapping.

Alt + E : Jugar como el oponente.

Alt + 2 : Alternar Modo 2 Jugadores Local.

Alt + D : Alternar Downscroll.

Alt + M : Ciclar modos de Middlescroll.

Interfaz y Visuales
Alt + S : Ciclar la posición del texto de puntuación.

Alt + L : Activar/Desactivar el fondo oscuro de las Strumlines.

Alt + V : Ocultar/Mostrar las notas del oponente.

Alt + O : Aumentar la opacidad del carril.

Alt + U : Ciclar el modo de los Popups.

Alt + P : Ciclar cuándo se mueven los Popups.

Alt + Y : Alternar el tipo de hitbox en móviles.

Cámara y Escenario
Alt + C : Activar/Desactivar el paneo libre de cámara.

Alt + H : Activar/Desactivar Showcase Mode (Oculta la UI).

Alt + X : Mostrar/Ocultar los marcadores de posición de personajes.

Alt + Z : Activar/Desactivar el Simple Mode.

Alt + Shift + Q : Mostrar los bordes de colisión de flechas.

Control de Tiempo y Canciones
Shift + R : Reiniciar la canción actual (Funciona siempre).

Alt + 1 : Reducir la velocidad en -0.1x.

Alt + 2 : Aumentar la velocidad en +0.1x.

Alt + 3 : Retroceder 10 segundos.

Alt + 4 : Adelantar 10 segundos.

Alt + 7 : Canción anterior en la Playlist.

Alt + 8 : Canción siguiente en la Playlist.
